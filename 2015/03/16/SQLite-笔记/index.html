SQLite 笔记

发表于 2015-03-16

#####1、SQLite数据操作语句示例：（汉语是要替换的东东）
　　查： select 字段名 from 表 where 字段名 比较符 字段值 and 字段名 比较符 字段值 limit 条数;
　　增：insert into 要插入数据的表 (字段名1,字段名2) values (字段值1，字段值2);
　　删：delete from 要删除数据的表 where 字段名 比较符 字段值;
　　改：update 要更新的表 set 字段名1=字段值1 where 字段名 比较符 字段值;

　　select 过滤列，where过滤行；

#####2、字符串应该严格用【单引号】引用，串内用单引号应该用2个单引号表示。

#####3、10进制数字直接写，16进制数字用 x’0000’ 表示，0000是16进制数，并且个数必须是偶数，因为SQLite是先写首位而不是先写末位。

#####4、SQLite不区分大小写，除了字符常量值（比如：’Mike’ 与 ‘mike’不同，name 与 NAME、from 与 FROM 相同）。

#####5、SQLite有五种本地类型integer、real、text、blob、null。（整形、实型、文本、二进制、空）
　　blob是Binary Large OBject的简写，即二进制大对象，存储图片等。

#####6、LIKE判定式大小写【不敏感】。
　　LIKE的通配符是“%”，表示0到若干个任意字符；
　　LIKE的单配符是下划线“_”，表示一个任意字符。

#####7、GLOB判定式大小写【敏感】。//glob是globbing的简写，通配的意思
　　GLOB的通配符是“*”，表示0到若干个任意字符；
　　GLOB的单配符是下划线“_”，表示一个任意字符。
